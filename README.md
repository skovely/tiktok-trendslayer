# tiktok-trendslayer
TikTok Trend Slayer 是一款专为跨境卖家和达人机构设计的 AI 选品助手。它不只搬运榜单，更通过深度神经网络分析流量波形的“突变点”，帮你挖掘那些尚未被大卖围猎的蓝海商品。

**【为什么选择 TikTok Trend Slayer？】**

在 TikTok 电商领域，领先 48 小时发现爆款意味着 10 倍的利润空间。传统的选品工具只告诉你“什么火了”，而 ***TikTok Trend Slayer*** 告诉你“什么即将火”以及“为什么火”。将原本需要 3 小时的手动刷榜和分析，缩短为 1 分钟的“选品简报”。


**【核心功能特性】**
* **🚀 黑马发现算法 (Hidden Gem Detection)***：实时监测 GMV 增长斜率。当某个商品在 24 小时内销量增速翻倍，且挂车达人数仍处于低位时，系统将自动触发“蓝海预警”。
* **🎭 视频病毒基因拆解***：AI 自动解析高转化视频的“黄金 3 秒”Hook、脚本结构与 BGM 情绪，为您提供 1:1 可复刻的爆款脚本公式。
* **👥 达人撮合雷达***：基于商品画像自动筛选最具带货潜力的 KOC，拒绝只看粉丝数，只看实战转化率。
* **📊 一键生成选品周报***：支持导出包含商品图、当前销量、预估利润、竞争程度及推荐话术的完整报告。

**👥【适用人群】**
***跨境卖家***：寻找高毛利、低竞争的 3C、美妆、家居黑马产品。
***MCN 机构***：快速为旗下达人匹配高转化 SKU，缩短测品周期。
***独立站玩家***：捕捉社媒流行趋势，实现流量套利。

**⌨️ 指令示例 (Sample Prompts)**
**1** **全局搜索**：@TikTok-Trend-Slayer 帮我找出过去 48 小时内在美国区 3C 类目下增长最快的 3 个黑马商品。
**2** **深度拆解**：针对这款 [商品名]，分析其最近 5 个爆火视频的脚本结构，并给我 3 个适合拍摄的 Hook。
**3** **达人匹配**：我想推一款磁吸充电宝，请列出 5 位近期带过同类产品且互动率超过 5% 的中小达人。
